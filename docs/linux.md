# linux

## Requirements

1. check if Python3 is installed and in PATH  
`which python3`
2. check if virtualenv is installed  
`pip3 list | grep virtualenv`


## Download the source

```bash
git clone https://github.com/cesargoncalves/python3-venv.git
```

## Setup dependencies

```bash
./run.sh --setup_dependencies
```

## Run the code

```bash
./run.sh
```
